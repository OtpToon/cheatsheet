Html is used to create the structure and contents of a webpage.

A tag always uses "<" and ">" to open. Most tags require a closing tag as well using "</" and ">".

Basefile

Every file should have a basic structure very similar to this:

<!DOCTYPE html>
<html lang="en">
    <head>
        <meta charset="UTF-8">
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
        <link rel="stylesheet" href="[path to stylesheet]">
        <title>[Title showing in the tab]</title>
    </head>
    <body>
    </body>
</html>

html-tag

Tag Name Description
<html> and </html> html Wraps around the whole document, letting the browser know what to show.

head-tag

Tag Name Description
<head> and </head> head Is used for metadata, to link fonts or stylesheets and to set the title
<link rel="" href=""> link Is used to link fonts or stylesheets. rel sets what is being linked and href sets the path to what is being linked
<title> and </title> title Sets the title visible in the tab

body-tag

Tag Name Description
<body> and </body> body The wrapper element for all content that is visible on the page
<nav> and </nav> Navigation Wrapper for the Navigation part of the website, usually combined with a unordered list
<div> and </div> Division Is used to divide the html into logical chunks to make working with it easier

content-tags

Tag Name Description
<h1> - </h1>
<h6> - </h6> 		Heading 1 - 6 Used for different kinds of headings with 1 being the biggest and 6 being the smallest
<p> and </p> Paragraph Used to put simple text on the website
<span> and <> Span Used to select a small subportion of text to change its properties
<a href="" target=""> and </a> Hyperlink Used to link to files or websites with href being the url or the path. target can be used to open the link in a new tab, window etc.
<img src="" alt=""> Image Used to show an image from a url or file using src. alt is used to display text if the image can't be displayed.
Can be used inside of a to create a Hyperlink.
<ul> and </ul> Unordered list Used to create a list of items that do not count up. Used wrapped around li
<ol> and </ol> Ordered list Used to create a list of items that does count up. Used wrapped around li
<li> and </li> List item Used to wrap each item inside ordered or unordered lists.

table-tag

Tag Name Description
<table> and </table> Table The wrapper element for all HTML tables.
<tr> and </tr> Table Head Used to define what each column of the table is showing
<tr> and </tr> Table Body Wraps around all elements of the table
<tr> and </tr> Table Row Used to wrap each row inside a table
<tr> and </tr> Table Data Used to wrap around the data displayed in a column in a row
<tr> and </tr> Table Foot Used if the last row in the table is supposed to have some conclusion and should be styled special
