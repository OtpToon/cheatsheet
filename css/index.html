CSS

CSS is used to style a website by giving different html-snippets different attributes.

Typical syntax for Css is:

[Identifier] {
[Attribute]: [Value];
...
}

Identifiers

The tag and class identifiers can be combined to more accuratly decide which elements should be changed

for a class within a tag:

[name] .[name2]{
...
}

for a tag inside a class:

.[name] [name2]{
...
}
Identifier Name Description
[name] tag Typical html-tags like h1, p, etc
.[name] class Marked in html with <tag class="[name]">.
#[name] id Marked in html with <tag id="[name]">.

Atributes

Attribute Name Valid Values Description
display Display block, inline, inline-block or none (invisible) The display value decides if the element is visible and how it interacts with neighboring elements
position Position static, absolute, fixed, relative, sticky The position value decides the location of the element within the users window.
z-index Z-index auto, number Z-index decides which element will be shown if multiple elements claim the same space.

margin Margin length, %, auto The margin defines the distance neighboring elements need to have from the border of this element.
Margin can be set using 1 (all sides), 2 (top/bottom, sides),3 (top, sides, bottom) or 4 (top, right, bottom, left) Values.
The distances can also be set on their own using margin-top/right/left/bottom.
padding Padding length, % The margin defines the distance from the content of the element to the border of the element.
Padding can be set using 1 (all sides), 2 (top/bottom, sides),3 (top, sides, bottom) or 4 (top, right, bottom, left) Values.
The distances can also be set on their own using padding-top/right/left/bottom.
