Bash

Bash usually works in all Linux and Mac Terminals. On Windows you'll need an extra Commandline like git Bash.

A "directory" is often refered to as "Folder" in common speach and you "specify" with a space " " and an "argument" after the command.

command argument

If there is spaces in the names of files or directories they need to be marked with "\" or surrounded with ' .

For example

cd This\ Folder/That\ Folder

or

cd 'This Folder'/'That Folder'

Command Name Description
pwd print working directory Prints out the path which your commands take as starting location
ls list Lists all directories and Files in your specified directory (if not specified as argument it will use current working directory)
cd change directory Changes your current working directory to specified directory
mkdir make directory Creates a new directory at the specified path. If only a name is given the new directory will be located in the current working directory. Can't be an already existing directory.
touch Touch Creates a new file at the specified path or updates the "Date modified" to now if the file already exists. Can't be the name of an existing directory.
